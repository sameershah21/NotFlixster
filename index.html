    Flixster like application

    Project maintained by sameershah21 Hosted on GitHub Pages — Theme by mattgraham

    Welcome to GitHub Pages.

    A simple MVC application built to replicate a Movie site like IMdb or Flixster. API calls a movie database API and loads the information on the page. It also caches information or movie names searched for frequently, so as to save http get calls. This is done via database first Entity Framework.
